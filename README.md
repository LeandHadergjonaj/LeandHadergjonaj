## Start Here

I just graduated from Royal Holloway and went straight into work as an AI / Data
Science Engineer here in London. The throughline in everything I build is automation:
if a person is doing something by hand more than twice, I'd rather spend the afternoon
making it run itself. That instinct turned into a habit of shipping the tools I wished
already existed — a usage HUD for my Claude sessions, an honest tracker for where my
hours actually go, a model for the London rental market. AI isn't a feature I bolt on
at the end; it's in the loop from the first commit. Open source by default, because the
useful version of a tool is the one other people can read and fork.

## Current Projects

**[claude-usage-hud](https://github.com/LeandHadergjonaj/claude-usage-hud)** — a SwiftUI
menu bar app that shows your Claude.ai session usage in real time, so you stop guessing
how much of your window is left.

**[focuslog](https://github.com/LeandHadergjonaj/focuslog)** — a passive macOS menu bar
app that quietly watches keyboard and mouse activity to build an honest picture of how
you actually spend time at the machine.

**[PROJECT](https://github.com/LeandHadergjonaj/PROJECT)** — applied ML on the Greater
London rental market: Ridge Regression over multi-source data, wrapped in a Streamlit
dashboard. My Royal Holloway final-year dissertation.

**Maroni** *(private)* — full-stack workforce time-tracking SaaS for SMBs: React Native
app with employee and admin roles, FastAPI backend, Next.js site, Supabase, Stripe, GPS
session tracking, and plan gating. Built and shipped solo.
